About

This site was created as a senior project under the direction of Dr. Cortens at Boise State University to function as a Truth Table Generator and grader for students looking to practice different logical statements and their respective evaluation. Please visit our help page for more information on the problem types and a brief explanation of logical syntax, semantics and grammar.

For Future Developers

We've created this site as Spring Boot Web application using ThymeLeaf view with Bootstrap. Below you can find resources that will help familiarize yourself with the framework. We recommend using Intellij (premium edition is free for students) or VSCode for IDE use.
